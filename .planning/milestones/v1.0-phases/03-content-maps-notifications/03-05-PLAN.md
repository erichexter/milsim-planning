Info sections endpoints (03-02):
- GET  /api/events/{eventId}/info-sections → [{ id, title, bodyMarkdown, order, attachments: [{id, friendlyName, contentType, fileSizeBytes}] }]
- POST /api/events/{eventId}/info-sections → { id, title, bodyMarkdown, order }
- PUT  /api/events/{eventId}/info-sections/{id} → 204
- DELETE /api/events/{eventId}/info-sections/{id} → 204
- PATCH /api/events/{eventId}/info-sections/reorder → 204 (body: { orderedIds: string[] })
- GET  /api/events/{eventId}/info-sections/{sectionId}/attachments/upload-url → { uploadId, presignedPutUrl, r2Key }
- POST /api/events/{eventId}/info-sections/{sectionId}/attachments/confirm → 201 (body: { r2Key, friendlyName, contentType, fileSizeBytes })

Map resources endpoints (03-03):
- POST /api/events/{eventId}/map-resources/external → { id, externalUrl, instructions, friendlyName }
- GET  /api/events/{eventId}/map-resources → [{ id, externalUrl, instructions, r2Key (null for list), friendlyName, contentType, order }]
- DELETE /api/events/{eventId}/map-resources/{id} → 204
- GET  /api/events/{eventId}/map-resources/{id}/upload-url → { uploadId, presignedPutUrl, r2Key }
- POST /api/events/{eventId}/map-resources/{id}/confirm → 204
- GET  /api/events/{eventId}/map-resources/{id}/download-url → { downloadUrl }

Notification blast endpoints (03-04):
- POST /api/events/{eventId}/notification-blasts → 202 { blastId, recipientCount }
- GET  /api/events/{eventId}/notification-blasts → [{ id, subject, sentAt, recipientCount }]
