```csharp
public class MapResource
{
    public Guid Id { get; set; }
    public Guid EventId { get; set; }
    public string? ExternalUrl { get; set; }   // set for external links; null for file resources
    public string? Instructions { get; set; }   // markdown
    public string? R2Key { get; set; }          // set for file resources; null for external links
    public string? FriendlyName { get; set; }   // display name
    public string? ContentType { get; set; }    // MIME type (file resources only)
    public int Order { get; set; }
    public Event Event { get; set; } = null!;
}
```

From milsim-platform/src/MilsimPlanning.Api/Services/FileService.cs:
```csharp
public interface IFileService
{
    UploadUrlResponse GenerateUploadUrl(Guid eventId, Guid resourceId, string contentType, string fileName);
    string GenerateDownloadUrl(string r2Key);
}
public record UploadUrlResponse(Guid UploadId, string PresignedPutUrl, string R2Key);
```

Note: MapResource supports two resource types:
- External link: ExternalUrl set, R2Key null
- File resource: R2Key set, ExternalUrl null
Both can have Instructions (markdown).

Allowed MIME types for map files (from FileService AllowedMimeTypes whitelist):
- application/pdf, image/jpeg, image/png, image/gif
- application/vnd.google-earth.kmz, application/vnd.google-earth.kml+xml, application/zip

CRITICAL — Pre-signed URL on demand (NOT on page load):
- DO NOT return download URLs in list endpoints
- Generate download URL ONLY when client calls the dedicated download-url endpoint
- Store only R2Key in DB — never the full URL

Established patterns from Phase 2:
- [Route("api/events/{eventId:guid}/map-resources")]
- [Authorize(Policy = "RequireFactionCommander")] for writes
- [Authorize(Policy = "RequirePlayer")] for reads (including player download)
- ScopeGuard.AssertEventAccess(_currentUser, eventId) as first line

    - CreateExternalLinkAsync: creates MapResource with ExternalUrl + optional Instructions; R2Key null
    - GetUploadUrlAsync: validates file size ≤ 10MB and MIME type; calls IFileService.GenerateUploadUrl; creates a pending MapResource (R2Key = null, FriendlyName set) to track the upload; returns uploadId + presignedPutUrl + r2Key
    - ConfirmFileUploadAsync: finds pending MapResource by uploadId, sets R2Key, ContentType, FileSizeBytes; validates R2Key matches what was generated
    - GetDownloadUrlAsync: generates pre-signed GET URL on demand (NEVER on list); throws NotFoundException if resource has no R2Key
    - DeleteMapResourceAsync: removes MapResource; throws NotFoundException if wrong event
    - ListMapResourcesAsync: returns ordered list (by Order field); does NOT include download URLs

    ```csharp
    // Models/Maps/ConfirmMapFileRequest.cs
    ```

    - ListMapResourcesAsync: order by Order ascending; select projection (exclude any URL generation)
